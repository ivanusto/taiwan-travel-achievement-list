# 台湾旅行アチーブメントシステム

🌐 **Languages / 多語言版本**: [繁體中文](README.md) | [English](README.en.md) | [日本語](README.ja.md) | [한국어](README.ko.md)

ゲームの **アチーブメントシステム**、**メトロデザインUI**、**トラベル手帳風イラスト** を融合させた、個人向けの台湾旅行記録・シェアカード生成プラットフォームです。

このシステムは完全にサーバーレスの静的ウェブサイトです。インストールやアカウント登録の必要はなく、ブラウザ上で直接チェックインを行い、旅行アチーブメントをアンロックできます。

---

## 🌐 オンライン使用URL

👉 **[台湾旅行アチーブメントシステムを開始する](https://ivanusto.github.io/taiwan-travel-achievement-list/)**

* 旅行の進捗はブラウザのローカルストレージ（LocalStorage）に自動保存されるため、プライバシー的にも安全です。
* 旅行の進跡をいつでも記録できるよう、このURLをブックマークに登録することをおすすめします！

---

## 🌟 主な機能

1. **Dashboard 統計パネル**: プレイヤーのレベル（アチーブメント10個解除ごとに1レベルアップ）と称号（例：`🌱 台湾ビギナー`、`✈ 台湾エクスプローラー`、`🏯 台湾旅行の神` など）を即座に計算します。
2. **22県市足跡マップ**: 台湾の地理をグリッド状に表現したインタラクティブマップ。クリックでチェックインし、レトロな赤インクのスタンプを押すことができます。地域ごとの達成度もリアルタイムに表示されます。
3. **Canvas 探索レーダーチャート**: 外部ライブラリを一切使わず、ネイティブのCanvasで10大カテゴリーの探索進捗を美しく描画します。
4. **100の厳選アチーブメント**: ビギナーから修羅まで6段階の難易度。解除するまで内容が見えない **Steam風の隠しアチーブメント** にも対応しており、ホバーや解鎖で驚きの解説が現れます。
5. **10の名誉バッジ**: `🚄 交通・鉄道達人`、`🍢 夜市グルメ王`、`🚢 離島トラベラー` など、マイルストーン達成によって自動解鎖されるコレクションバッジ。
6. **一クリック画像シェア (PNG)**: ニックネーム、アバター、レベル、レーダーチャート、バッジを統合した800x1200の高解像度シェアカードを自動合成してダウンロード。
7. **ダブルテーマ切替**: 電子ゲーム風のダークスタイルと、トラベルノート和紙風のライトスタイルを瞬時に切り替え可能。
8. **多言語対応**: 繁体字中国語、英語、日本語、韓国語の4言語に完全対応。
9. **バックアップと復元**: ローカルストレージに自動保存。JSON形式ファイルでの進捗のエクスポートおよびインポートに対応し、デバイス間での同期も容易です。
10. **完全サーバーレス**: 静的なHTMLファイル1枚のみで構成され、爆速で読み込み。データは100%端末ローカルに留まるためプライバシーも万全です。
