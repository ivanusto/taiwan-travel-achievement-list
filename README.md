# 台灣旅行成就系統

🌐 **Languages / 多語言版本**: [繁體中文](README.md) | [English](README.en.md) | [日本語](README.ja.md)

一個結合 **成就系統**、**磁貼美學** 與 **手帳插畫風格** 的個人台灣旅遊足跡打卡與分享圖卡生成平台。

本系統完全部署於雲端，玩家無需安裝任何程式或下載檔案，即可直接在瀏覽器中完成所有的打卡記錄，解鎖台灣旅行成就。

## 🌟 核心特色

1. **Dashboard 統計看板**: 即時計算玩家等級（每 10 個成就為 1 級）與稱號（例如：`🌱 台灣新手`、`✈ 台灣探索家`、`🏯 台灣旅行神` 等）。
2. **22 縣市足跡地圖**: 互動式台灣地理網格地圖，點擊可直接打卡並蓋上紅色拜訪印章，動態反饋分區探索率。
3. **Canvas 探索雷達圖**: 原生 Canvas 繪製，無任何外部第三方 JS 庫依賴，完美顯示 10 大分類的探索度。
4. **100 項成就資料庫**: 包含 Beginner 至 Shura 共計六大難度，且支持 **Steam 隱藏成就** 效果，解鎖時可看見驚喜描述。
5. **10 大榮譽徽章牆**: 自動判定如 `🚄 鐵道與交通達人`、`🍢 夜市饕客`、`🚢 跳島旅行家` 等徽章解鎖進度。
6. **一鍵分享圖卡 (PNG)**: Canvas 自動合成一張包含暱稱、頭像、等級、徽章牆與雷達圖的 800x1200 高解析度名片。
7. **雙主題切換**: 支持 Steam 電玩暗色風與手帳和紙淺色風。
8. **三語切換**: 支持 繁體中文、English、日本語 語系。
9. **備份與還原**: 本地 LocalStorage 自動存檔，並支持匯出與導入 JSON 進度，方便跨裝置同步。
10. **無任何後端**: 完全靜態的單一 HTML 檔案，載入極快，資料 100% 留存在本地端，確保使用者隱私。
