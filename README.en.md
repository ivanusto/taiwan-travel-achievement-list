# Taiwan Travel Achievement List

🌐 **Languages / 多語言版本**: [繁體中文](README.md) | [English](README.en.md) | [日本語](README.ja.md)

A personal Taiwan travel footprint tracker and sharing card generator that combines **achievement systems**, **metro design aesthetics**, and **travel notebook styles**.

This system is deployed entirely on the cloud. Players do not need to install any program or download any files; they can complete all check-ins directly in the browser.

## 🌐 Live URL

👉 **[Start Your Taiwan Travel Achievement Challenge Now](https://ivanusto.github.io/taiwan-travel-achievement-list/)**

* The project progress is automatically saved in your browser's local cache (LocalStorage), ensuring privacy and security.
* It is recommended to add this URL to your browser bookmarks for easy check-ins!

## 🌟 Core Features

1. **Dashboard Statistics Panel**: Instantly calculates player level (1 level per 10 achievements unlocked) and title titles (e.g., `🌱 Taiwan Novice`, `✈ Taiwan Explorer`, `🏯 Taiwan Travel Deity`, etc.).
2. **22 Counties footprint Map**: An interactive grid map of Taiwan's geography. Click to check in and stamp a red footprint, with dynamic region completion rates.
3. **Canvas Radar Chart**: Rendered with native Canvas, zero external JS library dependencies, perfectly showing the exploration levels of the 10 major categories.
4. **100 Curated Achievements**: Ranging from beginner to shura star levels, including **Steam Hidden Achievements** that reveal contents only upon hover or unlock.
5. **10 Honor Badges**: Automatically unlocks titles like `🚄 Transit Master`, `🍢 Night Market Foodie`, `🚢 Island Hopper`, etc., based on milestones.
6. **One-Click Share Card (PNG)**: Automatically compiles a high-resolution 800x1200 card containing name, avatar, level, badge wall, and radar chart on a canvas for sharing.
7. **Dual-Theme Toggle**: Supports dark gaming style and light notebook washi paper style.
8. **Tri-lingual Support**: Supports Traditional Chinese, English, and Japanese languages.
9. **Backup and Restore**: LocalStorage auto-save, supporting JSON progress export and import for cross-device syncing.
10. **Serverless Static Page**: A single pure HTML file loading extremely fast, keeping 100% of user data locally.
