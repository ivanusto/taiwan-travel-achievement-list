# 대만 여행 아치이브먼트 시스템

🌐 **Languages / 다국어 버전**: [繁體中文](README.md) | [English](README.en.md) | [日本語](README.ja.md) | [한국어](README.ko.md)

게임의 **아치이브먼트(업적) 시스템**, **메트로 디자인 UI**, **여행 수첩 일러스트 스타일**을 융합한 개인용 대만 여행 기록 및 공유 카드 생성 플랫폼입니다.

이 시스템은 완전히 서버리스로 동작하는 정적 웹사이트입니다. 별도의 설치나 가입이 필요 없으며, 브라우저에서 직접 체크인하고 여행 업적을 해제할 수 있습니다.

---

## 🌐 온라인 사용 URL

👉 **[지금 대만 여행 아치이브먼트 챌린지 시작하기](https://ivanusto.github.io/taiwan-travel-achievement-list/)**

* 여행 기록은 브라우저의 로컬 스토리지(LocalStorage)에 자동 저장되므로 개인정보 유출 우려 없이 안전합니다.
* 여행 발자취를 편리하게 기록할 수 있도록 이 URL을 브라우저 북마크에 등록하는 것을 추천합니다!

## 🌟 주요 특징

1. **대시보드 통계 패널**: 플레이어의 레벨(업적 10개 해제 시마다 1레벨 상승)과 칭호(예: `🌱 대만 초보자`, `✈ 대만 탐험가`, `🏯 대만 여행의 신` 등)를 실시간으로 계산해 보여줍니다.
2. **22개 시·현 발자국 지도**: 대만 지리 구성을 그리드 형태로 표현한 인터랙티브 지도입니다. 클릭 시 빨간색 방문 스탬프 애니메이션 효과와 함께 체크인되며, 지역별 탐험률을 실시간 반영합니다.
3. **Canvas 탐험 레이더 차트**: 외부 라이브러리 사용 없이 순수 HTML5 Canvas를 이용해 10대 분류별 탐험도를 직관적이고 아름답게 그려냅니다.
4. **100가지 엄선 업적**: 초보자부터 수라 난이도까지 6단계로 구성되어 있으며, 획득 전에 내용을 볼 수 없는 **Steam 스타일 숨겨진 업적** 기능을 완벽히 지원합니다.
5. **10가지 명예 배지 보관함**: `🚄 교통 철도 매니아`, `🍢 야시장 미식가`, `🚢 섬 탐험가` 등 특정 조건 달성 시 컬렉션에 자동으로 추가되는 배지입니다.
6. **한 번에 카드 공유 (PNG)**: 닉네임, 아바타, 레벨, 레이더 차트, 배지 벽면을 통합한 800x1200 고해상도 이미지 카드를 자동으로 합성하여 다운로드할 수 있습니다.
7. **듀얼 테마 지원**: 전자기기 감성의 다크 모드와 여행 수첩 감성의 라이트 모드 테마를 완벽하게 제공합니다.
8. **다국어 지원**: 繁體中文(번체 한국어), English(영어), 日本語(일본어), 한국어(Korean) 4개 언어를 완벽하게 교체하여 사용할 수 있습니다.
9. **백업 및 복구**: 로컬 스토리지 자동 저장 외에도 JSON 형식의 백업 파일 내보내기/가져오기 기능을 제공하여 기기 간 데이터 동기화가 편리합니다.
10. **완전한 서버리스 구성**: 단 하나의 HTML 파일로만 이루어진 정적 페이지로 매우 빠르게 로딩되며, 데이터가 100% 사용자의 브라우저 로컬 환경에만 상주하여 프라이버시가 안전하게 보호됩니다.
