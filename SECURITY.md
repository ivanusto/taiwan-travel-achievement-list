# Security Policy

## Supported Versions

Only the latest release version is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a potential security vulnerability in this project, please report it responsibly:

1. **Do NOT** open a public GitHub issue.
2. Email your findings to `ivanusto@gmail.com` or create a Private Vulnerability Advisory on GitHub.
3. Include details of the vulnerability, impact, and reproduction steps.

We will review your report within 48 hours and work on a fix promptly. Thank you for helping keep this project secure!
